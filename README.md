
# Bioproteg.com — Next.js (Vercel Ready)

## Requisitos
- Node.js 18+ (Vercel lo gestiona automáticamente)
- Cuenta en Vercel

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`
- `npm start`

## Variables de entorno (Vercel → Project Settings → Environment Variables)
- `SMTP_HOST` = smtp.zoho.com (o el que uses)
- `SMTP_PORT` = 465
- `SMTP_USER` = contacto@bioproteg.com
- `SMTP_PASS` = (clave de aplicación)
- `TO_EMAIL`  = contacto@bioproteg.com

## Deploy en Vercel
1. Sube este proyecto a GitHub o súbelo como ZIP en Vercel → New Project.
2. Configura las variables de entorno indicadas.
3. Deploy. Listo.

## Imágenes
Se ubicaron en `/public/img`. Puedes reemplazarlas.
